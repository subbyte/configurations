# Keyboard Configurations

### HHKB

#### 0. Mode: 001010
- Delete -> Backspace
- Alt <-> Mod

#### 1. (Left) Control -> Esc

#### 2. Esc -> (Right) Control

#### 3. (Right) Alt <-> (Left) Control

### Standard Keyboard

#### 1. Esc <-> CapsLock

#### 2. Backspace <-> Backslash

#### 3. (Right) Alt -> (Left) Control

#### 4. Grave <-> (Right) Control
